html 热刷新

1、安装raw-loader

npm install --save-dev raw-loader  
2、在webpack.config.js中配置raw-loader

    rules: [  
      ......  
      {  
         test: /\.(htm|html)$/,  
         use: [  
           'raw-loader'  
         ]  
      }  
      ......  
    ] 
3、在入口文件index.js文件中引入index.html文件

import '../template/index.html'
